Hi Sergey!

Yes, the latest one should be in the damaris directory.
Bugfixes, specific to a release, e.g. damaris-0.11should be in the
dedicated directory.

If there are major changes/developments there will be a release:
* a notification about the changes,
* a new directory, e.g. damaris-0.12, and
* a debian package built and tested.

Unfortunately there is no debian package repository because at berlios
it is mandatory to release larger files by their release system.
